- baselines包括对比的baseline论文源码，包括FPMC，GRU4Rec，BERT4Rec，SASRec等，服务器上有师兄的BERD及STEAM源码
- IUI对应论文稀疏注意力
- IDUI对应论文识别并删除
- ICUI对应论文识别并修正
- 数据：Yelp，Steam，CD 具体可阅读论文中的数据集介绍
- 运行：有很多xxmain.py 可直接run，config为本文配置
- 服务器运行：
  - nvidia-smi查看空闲显卡
  - 修改main文件指定显卡（如有必要）
  - nohup python3 -u 【main.py】>> 【./xxlog/logname.log】2>&1 & 
    - 进行代码运行并记录输出，输出为log文件
- 具体代码框架继承了孙亚童师兄的BERD代码框架，可先学习师兄的源码再来看这个，有助于理解本文代码。
